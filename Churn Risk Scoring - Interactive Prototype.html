Churn risk scoring prototype — India

Checking API...

Prepaid high-value customer churn, scored live by a FastAPI backend (api.py) running the actual trained pipeline (XGBoost, 143-field set, selected on PR-AUC and top-20% capture rather than raw ROC-AUC because only 8.6% of high-value customers churn). Trained on 30,011 real high-value customers from the published "Telecom Churn Case Study" dataset (India/SE Asia prepaid market). Start the API with uvicorn api:app --reload --port 8000 before scoring.
